# Security Limits

## This project DOES NOT claim:
- Complete protocol security proof
- Formal verification completeness
- Real-world deployment security

## QKD assumptions
- QKD is treated as an optional entropy source
- No unconditional security claim is made

## Implementation limits
- Prototype-level implementation
- Not hardened for production

## Scope
- Focus: reproducibility and verifiability
- Not: full cryptographic guarantees
